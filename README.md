# 🎬 Movie Review Sentiment & Insight Analyzer
**ISOM5240 Group 23 Project**  
**For Netflix Content Optimization**

## 📋 Project Overview
This is a Streamlit web application that combines two deep learning pipelines to analyze movie reviews:
1. **Sentiment Analysis**: Fine-tuned DistilBERT model classifies reviews as POSITIVE or NEGATIVE with 90.15% accuracy
2. **Text Summarization**: BART-large-CNN model generates concise 1-2 sentence key insights from long reviews
